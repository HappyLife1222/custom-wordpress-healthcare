== Description ==

= Français =

WPS Child Theme Generator est un outil avancé pour créer facilement autant de thèmes enfants que nécessaire. Il est livré avec de nombreuses options qui peuvent être ajoutées à votre functions.php sans avoir besoin de le modifier. Vous pouvez également ajouter votre propre CSS dans style.css.

Options

1. Ajoutez votre capture d'écran de WordPress
2. Supprimer les tailles d'image
3. Ajouter une taille d'image
4. Supprimer les widgets
5. Activer le shortcode dans les widgets
6. Modifier l'URL du résultat de la recherche
7. Ajouter un type de publication aux résultats de recherche
8. Ajoutez des colonnes dans la table de liste d'admin (vignette et ID) pour le type de publication choisi
9. Retirer les "post type support"
10. Ajouter CSS à style.css

Creation behaviour:

* paquet de téléchargement
* thème enfant zippé et disponible dans wp-content/themes
* envoi par email

WPS Child Theme Generator is an advanced tools to easily create as many child theme as you need. It comes with many options that can be add to your functions.php without the need to edit it. You can also add your own CSS in style.css.

Options

1. Add your screenshot from WordPress Uploader
2. Remove image sizes
3. Add image size
4. Remove widgets
5. Enable shortcode in widgets
6. Modify search result URL
7. Add post type to search results
8. Add columns in admin list table (thumbnail and ID) for choosen post type
9. Remove post type supports
10. Add CSS to style.css

Creation behaviour:

* download package
* child theme is zip and available in wp-content/themes

= Français =

3. Recherchez ce plugin, téléchargez-le et activez-le.

= English =
1. Go to Plugins › Add New.
2. Search for *WPS Child Theme Generator*.
3. Look for this plugin, download and activate it.
